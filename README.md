Mulesoft Cert training

Resouces
* Console URL : http://127.0.0.1:8081/console/
* API URL : http://127.0.0.1:8081/api/crypto

Routes
* /crypto - GET ALL info from coincap.io front page
* /crypto/page/{coin} - GET coin object by coin name. Ex. BTC
